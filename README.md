# Build Net

[![Python CI](https://img.shields.io/github/actions/workflow/status/denezt/build-net/python-ci.yml?label=Build\&logo=github)](https://github.com/denezt/build-net/actions) [![PyPI Version](https://img.shields.io/pypi/v/metadata-flow?color=blue\&logo=pypi)](https://pypi.org/project/metadata-flow/)

An automation toolkit for managing distributed project metadata. Combines data processing power for enterprise-scale metadata operations.

### Getting Started:

``` sh
$ uv init
```

``` sh
$ uv sync
```

### Run System
``` sh
$ sudo uv run main.py
```
